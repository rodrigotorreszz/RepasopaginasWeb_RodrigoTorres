Preguntas de repaso

¿Qué es HTML y cuál es su función principal en el desarrollo web?
Es el lenguaje estándar para crear y estructurar páginas web se basa en etiquetas y elementos.
¿Qué son las etiquetas HTML y cómo se utilizan?
HTML define elementos específicos y su presentación en la página mediante etiquetas que comienzan y terminan con símbolos especiales.
Proporciona ejemplos de algunas de las etiquetas HTML más comunes.
  • !DOCTYPE: Se utiliza para especificar la versión de HTML que se está utilizando en la página.
  • html: Define el inicio y el final de la página web.
  • head: Define la sección de encabezado de la página, donde se incluyen elementos como el título de la página, metaetiquetas, scripts, entre otros.
¿Qué es CSS y cómo se utiliza para dar estilo a una página web?
CSS, o "hoja de estilo en cascada", es un lenguaje que define el diseño de páginas web, complementando el contenido proporcionado por HTML.
¿Cuáles son las propiedades más importantes de CSS?
text-align, text-decoration, text-transform, text-indent, line-height, letter-spacing y word-spacing

¿Qué es el diseño responsivo y cómo se logra con CSS?
Un diseño web receptivo permite que las páginas web se adapten a varios tamaños y dispositivos de pantalla. Se usa HTML y CSS para ajustar el tamaño, ocultar, reducir, expandir o mover el contenido de manera adecuada en cualquier dispositivo.
¿Qué es JavaScript y cómo se integra en el desarrollo web?
JavaScript es un lenguaje de programación que se usa para crear sitios web interactivos. Se aplica en el desarrollo web, especialmente cuando se utiliza con documentos HTML.
Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).
JavaScript interactúa con el DOM para cambiar la dinámica de las páginas web en HTML y CSS. Esto ayuda a crear aplicaciones web que sean interactivas y respondan a las acciones del usuario.
Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.
  • Formularios interactivos
  • Animaciones de menú
  • Actualizaciones de contenido dinámico
¿Qué es Git y por qué es importante para el desarrollo web?
Esto ayuda a los desarrolladores web a trabajar juntos en proyectos, seguir los cambios y retroceder a versiones anteriores si es necesario. También favorece el trabajo en equipo, permitiendo que varios desarrolladores trabajen al mismo tiempo en diferentes funciones.
Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web.
GIT es un sistema que ayuda a los desarrolladores de software a seguir su código y las diferentes versiones de proyectos en los que trabajan en equipo. Es una herramienta útil para el desarrollo web.
¿Qué es GitHub y cómo facilita la colaboración en proyectos web?
Es un servicio en la nube con un sistema de administración de versiones llamado Git. Permite a los desarrolladores trabajar juntos y hacer cambios en proyectos compartidos, mientras se mantiene un registro detallado del progreso.
